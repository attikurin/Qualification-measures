MOS資格の概要

MOS（Microsoft Office Specialist）は、マイクロソフト社が公式に認定する国際資格で、 WordやExcelなどのMicrosoft Office製品の利用スキルを証明します。 世界的に認知されており、累計受験者数は国内だけで520万人を超えています。

基本関数（アソシエイト）

SUM

合計を求める

=SUM(A1:A10)
AVERAGE

平均を求める

=AVERAGE(A1:A10)
COUNT

数値の個数

=COUNT(A1:A10)
MAX / MIN

最大値 / 最小値

=MAX(A1:A10)
IF

条件分岐

=IF(A1>100,"合格","不合格")
VLOOKUP

垂直検索

=VLOOKUP(A1,B:D,2,FALSE)
LEFT / RIGHT / MID

文字列抽出

=LEFT(A1,3)
CONCATENATE

文字列結合

=CONCATENATE(A1,B1)

高度な関数（エキスパート）

SUMIF / SUMIFS

条件付き合計

=SUMIF(A:A,">100",B:B)
COUNTIF / COUNTIFS

条件付きカウント

=COUNTIF(A:A,"合格")
IFERROR

エラー処理

=IFERROR(A1/B1,0)
INDEX / MATCH

高度な検索

=INDEX(B:B,MATCH(A1,A:A,0))

参照形式

  • 相対参照：A1（コピー時に自動調整）
  • 絶対参照：$A$1（固定）
  • 複合参照：$A1、A$1（行または列のみ固定）
絶対暗記： VLOOKUP、SUMIF、COUNTIF、IFERRORは必須。絶対参照$の使い分けも重要。
