試験概要

Webクリエイター能力認定試験は、株式会社サーティファイのWeb利用・技術認定委員会が主催する、 Webサイト制作に必要なHTMLとCSSのコーディング能力を測定・評価する民間資格試験です。 W3C規格に完全準拠したセマンティックなマークアップスキルが評価されます。

試験の基本情報

試験形式 実技試験のみ（HTMLとCSSのコーディング）
試験時間 60分（2025年11月1日以降）
出題内容 1テーマ（4ページ程度のHTMLとCSSの作成）
受験料 6,100円（税込）※2026年4月より6,400円
受験資格 学歴・年齢等の制限なし（誰でも受験可能）
試験方式 リモートWebテスト、随時試験、公開試験
累計受験者数 200,460名（2025年3月31日時点）

HTML5対応版への移行

Webクリエイター能力認定試験は、2018年よりHTML4.01対応版からHTML5対応版へ移行しました。 現在実施されている試験はすべてHTML5対応版です。

  • HTML5対応版：最新のWeb標準に準拠したHTML5とCSS3の知識・スキルを評価
  • セマンティックWeb：意味を持つ適切なマークアップを重視
  • レスポンシブデザイン：マルチデバイス対応の基礎知識を含む

試験区分

スタンダード（初級）

本講座の対象
  • セマンティックWebを理解し、HTML5をマークアップできる
  • CSSを用いてHTMLの構造を維持しつつ、Webページのデザインやレイアウトを表現できる
  • 基本的なWeb制作スキルの証明

エキスパート（上級）

上級
  • レイアウト手法や色彩設計等、ユーザビリティやアクセシビリティを考慮したWebデザイン
  • スクリプトを用いた動きのあるWebページ表示
  • マルチデバイス対応、新規サイト構築

出題範囲

試験範囲は大きく6つのカテゴリーに分かれています。 HTMLとCSSの基礎知識から、Webサイトの構成設計、ユーザビリティ、アクセシビリティまで幅広く出題されます。

制作環境

  • ファイルの操作（保存、複製、拡張子、ファイル名）
  • テキスト／ソースの操作（コピー&ペースト、入力、変更、削除）
  • ブラウザー／ドメイン（ブラウザー名、レンダリングエンジン、URL、スキーム）

ファイルの種類

  • HTML／CSS言語の特徴
  • 構造とデザインの分離
  • 画像（GIF／JPEG／PNGファイルの特徴、ビットマップ形式）

Webサイトの構成と設計

  • ページ構成（基本ページ、トップページ、テキスト・画像ページ、テーブル、フォーム）
  • レイアウト／パーツ設計（ヘッダー、フッター、ナビゲーション、コンテンツ領域）
  • バナー／ボタン

ユーザビリティ／アクセシビリティ

  • タイトル／見出しの統一（タイトルと見出しのルール）
  • 文字色と背景色（コントラスト）

HTML5

  • セマンティック／コンテンツモデル／カテゴリー
  • HTML5の概念と基本構造
  • 文字参照／実体参照（&lt; &gt; &amp; &copy;等）、コメント
  • 文書型宣言、html/head/body/title要素、文字エンコード（UTF-8）
  • 外部スタイルシート（link要素／CSSファイルの読み込み）
  • グローバル属性：id, class, lang, title, style等
  • 基本的な要素（div, span, h1-h6, p, br, strong, small, address）
  • リスト（ul, ol, li, dl, dt, dd）
  • ハイパーリンク（a要素と関連する属性：href, target）
  • 画像（img要素と関連する属性：src, alt, width, height）
  • テーブル（table, tr, td, th, caption, colspan, rowspan）
  • フォーム（form, input, textarea, label, button）
  • セクション（header, footer, article, section, nav, aside, main）

CSS3

  • 基本構文（@charset, コメント, @import）
  • スタイルの継承と優先順位（詳細度、カスケード）
  • セレクター（要素、class、id、疑似クラス:hover/:focus、子孫、子、隣接）
  • 単位：px, em, rem, %, vw, vh, pt等
  • ボックスモデル（margin, border, padding, width, height, box-sizing）
  • フォント（font-family, font-size, font-weight, font-style, color）
  • テキスト（text-align, line-height, letter-spacing, text-decoration）
  • 背景（background-color, background-image, background-size, background-position）
  • リスト（list-style-type, list-style-image, list-style-position）
  • テーブル（border-collapse, vertical-align, border-spacing）
  • ボックス（display, float, clear, position, z-index, overflow）
  • レイアウト：Flexbox（display: flex, justify-content, align-items等）
  • メディアクエリ：@media screen and (max-width: ...)

HTML/CSS攻略法

試験合格のための重要ポイントと効果的な学習戦略をまとめました。 HTMLとCSSの基本を押さえ、実践的なコーディングスキルを身につけましょう。

HTML攻略のポイント

1

セマンティックなマークアップ

意味を持つ適切な要素を選択することが重要です。

<header>, <nav>, <main>, <article>, <section>, <footer>
2

正しい文書構造

DOCTYPE宣言、html、head、bodyの基本構造を正確に記述します。

<!DOCTYPE html>
<html lang="ja">
3

属性の正しい使用

id、class、alt、href、srcなどの属性を適切に使い分けます。

<img src="image.jpg" alt="説明文">

CSS攻略のポイント

1

セレクターの理解

要素、class、id、疑似クラスなどのセレクターを使い分けます。

p { } .class { } #id { } a:hover { }
2

ボックスモデルの把握

margin、border、padding、widthの関係性を正確に理解します。

box-sizing: border-box;
3

レイアウト手法

float、position、flexboxなどのレイアウト方法を習得します。

display: flex;
justify-content: center;

模擬試験で仕上げ

  • 時間を測って模擬試験
  • 間違えた箇所を復習
  • 3回以上繰り返す
  • 安定して合格点を取れるまで練習

学習のコツ

反復学習

同じ内容を繰り返し学習することで定着率が上がります。1周目は全体把握、2周目で詳細、3周目で応用を意識しましょう。

エラーから学ぶ

エラーは成長のチャンス。エラーメッセージを読み、原因を特定し、修正方法を学びましょう。同じミスをしないよう記録を残すと効果的です。

実際のWebサイトを参考にする

好きなWebサイトをブラウザの開発者ツール（F12キー）で確認し、構造やスタイルを分析してみましょう。実践的な学びが得られます。

定期的な復習

学習直後、翌日、1週間後、試験前のタイミングで復習すると記憶が定着します。エビングハウスの忘却曲線を意識しましょう。

学習時の注意点

  • 完璧主義にならない：すべてを覚えようとせず、試験範囲に集中しましょう。
  • 写経だけで終わらない：コードを理解しながら書き、変更を加えて実験しましょう。
  • 環境構築に時間をかけすぎない：Visual Studio Codeなどシンプルなエディターで十分です。
  • 孤立しない：分からないことは検索し、コミュニティで質問し、学習仲間を作りましょう。
