# 試験対策講座ポータル

## 📖 プロジェクト概要

**試験対策講座ポータル**は、各種資格試験の合格を目指す学習者を支援するWebサイトです。現在は**ITパスポート試験**と**生成AIパスポート試験**の完全対策講座を提供しており、今後は他の資格講座を順次追加予定です。

## 🎯 プロジェクトの目標

- 体系的な学習コンテンツで資格試験合格をサポート
- 効率的な学習方法と頻出ポイントの提供
- ユーザーフレンドリーで見やすいインターフェース
- 今後の講座追加に対応できる拡張性の高い設計

    - 2進数・16進数の変換

- **学習方法セクション**:

- **ページトップへ戻るボタン**: 固定配置

### 3. 生成AIパスポート試験 完全対策講座ページ（genai.html）✨ NEW
- **ページヒーロー**: 講座タイトルと概要（緑系グラデーション）
- **目次ナビゲーション**: 各セクションへのスムーズスクロール（折りたたみ可能）
- **試験概要セクション**:
  - 生成AIパスポート試験とは（GUGA実施）
  - 試験の基本情報（60分・60問・IBT方式）
  - 受験料（一般11,000円、学生5,500円）
  - IBT方式の特徴（年5回実施）
  - 資格取得のメリット
- **出題範囲セクション**:
  - 5章の出題分野構成（具体的な出題数）
    - 第1章: AI基礎（約10～15問）- ダートマス会議、弱いAI/強いAI、機械学習4種類
    - 第2章: 生成AI（約10～15問）- CNN/VAE/GAN、Transformer、GPTモデルの進化、ハルシネーション
    - 第3章: AI動向（約10～15問）- RAG、AIエージェント、MCP、ディープフェイク
    - 第4章: リテラシー・法律（約15～20問）【最重要】- セキュリティ、AI新法、個人情報保護、著作権
    - 第5章: プロンプト技術（約10～15問）- LM/LLM、ハイパーパラメータ、プロンプトの4要素
- **合格基準セクション**:
  - 正答率80%以上（60問中48問正解）
  - 分野別基準なし（総合評価のみ）
  - 合格ライン早見表
- **勉強時間セクション**:
  - AI初学者：40～60時間（約1.5～2ヶ月）

  - 学習期間の設定例（2ヶ月プラン、1ヶ月プラン）
- **分野別攻略法セクション**:
